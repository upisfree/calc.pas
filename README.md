calc.pas
========

A very simple calculator with 23 lines. It can calculate the numbers of up to 32000 with 4 actions(+, -, *, /). It weighs 4 kb.

How to use:
<ol>
  <li>write first number;</li>
  <li>write second number;</li>
  <li>write action (+, -, *, /)</li>
  <li>have result;</li>
  <li>happiness.</li>
</ol>

It work on old systems, of course.

It's simple!
